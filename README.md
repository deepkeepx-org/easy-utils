# easy-utils [![License](http://img.shields.io/:license-MIT-brightgreen.svg)](https://github.com/deepkeepx-org/easy-utils/LICENSE)

### About

This is a general-purpose method library for Java.

### Requirement

jdk version 1.8

### Quick Start

```xml
<dependency>
    <groupId>com.deepkeepx.jar</groupId>
    <artifactId>easy-utils</artifactId>
    <version>1.0.0</version>
</dependency>
```
